4. ¿Cual es la diferencia entre un IDE y un editor de texto?

A) El IDE es mas avanzado que el editor de texto y ofrece funciones adicionales como depuración y autocompletado.

5. Escriba los comandos para realizar una actualización en un repositorio al cual usted tiene acceso. 
Recuerde que el orden afectaría que pueda dejar por fuera una actualización de otro usuario del mismo repositorio, perdiendo información.

1. git status -s 
2. git pull origin master
3. git add . 
4. git commit -m
5. git push origin master
